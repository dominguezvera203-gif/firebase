# Firebase Notes App

## Overview

Firebase Notes App is a simple mobile application built using React Native, Expo, and Firebase. The app allows users to create an account, log in, and manage their personal notes in real time.

This project was created to demonstrate the integration of Firebase services in a mobile application, including Authentication and Firestore Database.

---

## Features

- User Registration
- User Login
- User Logout
- Add Notes
- View Notes
- Edit Notes
- Delete Notes
- Real-time Database Updates

---

## Technologies Used

- React Native
- Expo
- Firebase Authentication
- Firebase Firestore
- React Navigation

---

## Application Screens

### Login Screen
Users can log in using their registered email and password.

### Register Screen
New users can create an account using email and password authentication.

### Home Screen
Users can:
- Add new notes
- Edit existing notes
- Delete notes
- View notes in real time
- Logout from the application

---

## Purpose of the Project

The purpose of this project is to help students understand how Backend-as-a-Service (BaaS) works using Firebase. It demonstrates basic mobile app development concepts such as authentication, database integration, CRUD operations, and real-time updates.

---

## CRUD Operations

The application supports the following CRUD functionalities:

- Create → Add new notes
- Read → Display notes from Firestore
- Update → Edit existing notes
- Delete → Remove notes from the database
